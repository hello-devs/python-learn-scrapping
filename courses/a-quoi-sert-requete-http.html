À quoi sert une requête HTTP ?

Contexte :

Python est un langage informatique « universel », c'est-à-dire qu'il est utilisé dans presque tous les domaines informatiques. Et notamment pour le back-end de sites internet. C'est à ce domaine que nous allons plus particulièrement nous intéresser dans ce cours. En effet, tous les sites internet dialoguent avec leur serveur par le protocole HTTP à l'aide de requêtes. HTTP signifie « Hypertext Transfer Protocol ». Une requête, c'est une question posée ici à un serveur et qui attend une réponse de ce dernier.

Mais qu'est-ce qu'une requête HTTP ?

Une requête HTTP « Hypertext Transfert Protocol » c'est-à-dire (protocole de transfert par texte ayant des liens vers d'autres documents) est une question posée par votre terminal (téléphone, ordinateur ou autres tablettes depuis un explorateur internet) ; par exemple dès que vous tapez www.ditesco.fr ou simplement ditesco dans la barre de recherche et Google, (car au final c'est lui qui reçoit cette demande) vous renvoie une liste de choix où il a trouvé le mot ditesco. Ensuite, à chaque fois que vous cliquez sur un menu ou un mot en hypertexte correspondant à une URL ou URI une requête est envoyée et Google ou le site sur lequel vous êtes connecté renvoie la réponse.

Définition :

HTTP soit « HyperText Transfer Protocol » c'est-à-dire Protocole de Transfert par Texte sensible ayant des Hyperliens vers d'autres documents, autrement dit des textes sur lesquels on clique pour naviguer vers d'autres documents ou pages internet.

Complément :

Qui parle avec qui ? Quel chemin prendre pour rejoindre le serveur qui nous retournera la réponse à notre requête ?

Une requête http ne peut se faire que si la connexion est établie entre l'émetteur de la requête et le destinataire.

Un peu de vocabulaire

Une requête c'est une demande de dialogue avec le serveur. En effet une requête peut faire une demande d'information mais peut aussi envoyer des informations pour modifier une ressource.

La requête fait une demande d'obtention d'information ou ressource ou une demande d'envoi d'information vers une ressource dans le but de la modifier.

Nous allons utiliser les mots « demander » « obtenir » « poster » « mettre » « entête » « tracer ou suivre » « options » « effacer » en anglais de python cela donne : « request » « get » « post » « put » « head » « trace » « option » « delete ».

Méthode :

Le processus d'une requête peut se résumer ainsi : établissement de la liaison → demande → obtention (ou) poster(ou) tracer (ou récupérer) l'entête → fermer la liaison.

Dans ce cours nous ne traiterons que la partie en gras.
