Bases de Numpy

Contexte :

Dans ce cours, nous apprendrons à utiliser les diverses fonctionnalités de la librairie Numpy. En particulier nous verrons comment définir, manipuler et effectuer des opérations sur les ndarrays, objet central de la librairie. La valeur ajoutée de Numpy par rapport au noyau standard de python se situe au niveau de la rapidité des calculs, de la facilité d'implémentation, ainsi que de l'économie d'espaces mémoire. Cette librairie est incontournable pour quiconque souhaite se lancer dans un projet d'intelligence artificielle ou de Data Science.

Pour une meilleure fluidité et une meilleure compréhension, nous illustrerons chaque exemple à l'aide de code. Cela vous permettra de tester directement les commandes dans votre environnement python.

Introduction

Fondamental :

Dans ce cours nous allons apprendre à utiliser la librairie Numpy.

Numpy est une librairie de python conçue pour manipuler efficacement les tableaux multidimensionnels. Les commandes sont simples et les calculs sont optimisés et rapides.

Numpy est utilisée dans de nombreux domaines, c'est notamment une librairie référence en Data Science qui exige la manipulation de bases de données volumineuses. Elle est écrite majoritairement en C et en python.

Comment définir un ndarray

L'élément clé de la librairie Numpy est l'objet ndarray. L'ensemble de la librairie est conçu autour de la manipulation de ces objets. Un ndarray est un tableau en n dimensions, contrairement aux listes python classiques, les dimensions du tableau sont définies de manière explicite.

Attention à ne pas confondre un ndarrray avec un array standard. En effet un array désigne n'importe quel objet itérable de python (liste, tuple, ndarray, etc.). Autrement dit, un ndarray est un type d'array spécifique à la librairie numpy.
